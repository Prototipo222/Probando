Documento OACI 4444

El Doc 4444 de OACI (PANS-ATM) define los procedimientos de navegación aérea y el formato del plan de vuelo.

¿Qué es?

El Doc 4444 es el Manual de Procedimientos para los Servicios de Navegación Aérea. Establece formatos, reglas y prácticas recomendadas para planes de vuelo y servicios ATS.

¿Para qué sirve?

Sirve para homogeneizar los planes de vuelo, asegurar la interoperabilidad entre estados y facilitar la coordinación entre unidades ATC de distintos países.

Importancia

Es la referencia internacional para la presentación de FPL, revisiones de ruta, coordenadas y datos de comunicaciones.
